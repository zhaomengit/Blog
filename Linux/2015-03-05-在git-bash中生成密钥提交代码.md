https://help.github.com/articles/generating-ssh-keys/
###注意:

在第3步中有个命令是这样的

    # start the ssh-agent in the background
    eval 'ssh-agent -s'
    # Agent pid 59566


在windows下应该`eval 'ssh-agent -s'`这个命令是执行失败的，理应为

    eval `ssh-agent -s`

单引号 <em>'</em> 转换成 反引号 <em>`</em>
